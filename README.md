
# flex_slu_app

Refactored Streamlit app for SLU verification of reinforced concrete sections.

## Run
1. Create virtual environment and install requirements:
```bash
python -m venv venv
source venv/bin/activate   # or venv\Scripts\activate on Windows
pip install -r requirements.txt
```

2. Run Streamlit:
```bash
streamlit run app.py
```

This project was generated automatically from an existing monolithic script.
Please review formulas and checks before using for real designs.
